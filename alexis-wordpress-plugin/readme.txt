Alexis is a new Wordpress Admin Theme tested on Wordpress 3.5+ This is a work in progress. Simply download the plugin folder, place into your /wp-content/plugins folder, and then activate it.

== Installation ==

1. Upload the folder titled, 'alexis-wordpress-plugin' to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. All Done!
